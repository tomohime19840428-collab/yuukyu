🗑 申請データのリセット

📧 メール通知設定
有給申請があったときに指定アドレスへメールで通知します。EmailJS（無料）を使用します。
📋 初期設定の手順
1. emailjs.com で無料アカウントを作成
2. 「Email Services」でGmailなどのメールサービスを追加 → Service ID をコピー
3. 「Email Templates」でテンプレートを作成し、本文に下記の変数を使用：
   {{to_email}} 宛先   {{applicant}} 申請者名   {{details}} 申請内容
   {{total_days}} 合計日数   {{submitted_at}} 申請日時
4. テンプレートの Template ID をコピー
5. 「Account」→「General」の Public Key をコピー
6. 下記フォームに入力して「保存」をクリック

📖 使い方ガイド
1
メンバーを追加する
「メンバー管理」でチームの全員を追加します。管理者は申請の承認ができます。名前・付与日数・役割を入力してください。
2
右上で自分の名前を選ぶ
画面右上のプルダウンから自分の名前を選びます。これで自分の有給状況が表示されます。
3
「申請する」タブから申請
日程・種別・理由を入力して申請します。申請後は管理者に承認してもらいます。
4
管理者が「承認」タブで処理
管理者は「承認」タブで申請一覧を確認し、承認または却下します。承認後に残日数が自動で更新されます。
⚠
データについて
データはこのブラウザに保存されます。別のPCや別のブラウザでは共有されません。「データをエクスポート」でバックアップをとることをお勧めします。
